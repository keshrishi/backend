# HouseMate Backend

This is the mock backend for the HouseMate application, powered by JSON Server.

## Prerequisites

- Node.js installed.

## Setup

1.  Navigate to this directory:
    ```bash
    cd backend
    ```
2.  Install dependencies:
    ```bash
    npm install
    ```

## Running the Server

Start the JSON server with:

```bash
npm start
```

The server will be available at `http://localhost:3000`.
